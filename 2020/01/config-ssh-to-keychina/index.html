将生成的ssh 私钥添加到 Mac 的keychain 中

1

$ ssh-add -K .ssh/is_rsa

将登录信息配置到.ssh/config中

1
2
3
4
5
6

$ touch ~/.ssh/config
$ vim ~/.ssh/config
# edit text
Host myvm
  Hostname ip
  User user

保存之后就可以使用如下命令快捷登录服务器了

1

$ ssh myvm
