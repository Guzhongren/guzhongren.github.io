## 生成模板文章

```zsh
$ # make post=[article_name].zh.md
$ make post=article.zh.md
```

## 预览

```zsh
$ make pre
```

## 提交修改

```zsh
$ # make commit message=提交信息
$ make commit message=提交信息
```

## 发布

```zsh
$ make pub
```
